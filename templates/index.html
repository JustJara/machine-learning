What functionalities does this web application have?




1 Load file into memory

In this functionality, the user is expected to upload a file from their device. Once uploaded, the system will process the file and extract the data, which will be stored in an in-memory dataframe. This dataframe allows efficient data manipulation and analysis.

2 Classify dataset

In this functionality, the user is expected to upload a dataframe with the data to be classified and specify the desired number of clusters. The system will process this data and generate the corresponding clusters.
